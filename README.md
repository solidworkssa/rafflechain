# RaffleChain

Provably fair on-chain raffle system.

## Features
- Multi-chain support (Base & Stacks)
- Secure Contract Logic
- React/Next.js Frontend

## Contracts
- Base: `contracts/base/src/RaffleChain.sol`
- Stacks: `contracts/stacks/contracts/rafflechain.clar`

## Setup
```bash
pnpm install
pnpm dev
```
